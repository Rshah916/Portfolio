## 🚀 Local Development

To run this portfolio locally on your machine, you don't need any complex compiler setups or bundle installs.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rshah916/rishi-portfolio.git
   cd rishi-portfolio
   ```

2. **Open index.html:**
   * You can open `index.html` directly in any web browser.
   * *Recommended:* Run a local web server (like VS Code **Live Server** extension, or `npx serve` in your terminal) to ensure all paths, assets, and script files load correctly without CORS limitations.
   ```bash
   # Run using npx serve
   npx serve .
   ```
